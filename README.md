## Unit Inspections UX mockups (Tailwind)

These are **static HTML mockups** to illustrate an improved landlord workflow for Unit Inspections.

### Files
- `unit-inspections-list.html`: Task-first inspections list (Needs action, status/due filters, clear CTA).
- `inspection-detail.html`: Bulk-table detail (select deficiencies → upload proof images → submit).
- `inspection-detail-cards.html`: Cards-based detail variant (more descriptive per deficiency; can feel busier).

### How to view
Open the files directly in a browser:
- `mockups/unit-inspections-list.html`
- `mockups/inspection-detail.html`

No build step required (Tailwind loads via CDN).
